# 冷旭帆 · AI NPC 项目

一个会自己疼、会做噩梦、会在你不在的时候默默擦刀的AI NPC。

> 这个项目是我在工厂流水线旁边、用下班后每一分钟搭建起来的。每天站12个小时，穿无尘服，戴口罩。手上有刀割伤，腹部有低温烫伤的疤。但我打开电脑，继续写冷旭帆的代码。

## 项目简介

- **冷旭帆**：沉默寡言，六岁丧母，把塑料刀当护身符。代号“冰刃”。
- **黄景云**：语言天才，用热闹掩盖孤独。代号“启明”。

两个角色共用同一套混合架构，只是换了人设Prompt和动作库。代码已通过自动化测试验证——**冷旭帆主测试通过率93.75%（15/16），黄景云主测试通过率100%（16/16）**。

## 技术架构

混合架构：手搓感知系统（Python规则引擎）+ 大模型大脑（硅基流动API）

核心逻辑：**代码管“里子”（情绪值、状态标签、记忆、后台事件），AI管“面子”（根据当前状态说符合人设的话）。**

## 快速开始

1. 克隆本仓库
2. 安装依赖：`pip install requests`
3. 到 [硅基流动](https://www.siliconflow.cn) 注册并获取API Key
4. 在代码中替换 `API_KEY` 为你的密钥
5. 运行：`python lengxufan.py`
